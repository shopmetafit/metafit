                         WHAT BLUE DART WILL GIVE YOU

═══════════════════════════════════════════════════════════════════════════════

Blue Dart sends you 5 things:

  1. API_KEY              (your unique key)
  2. API_SECRET           (your unique secret)
  3. LOGIN_ID             (your login ID)
  4. LICENSE_KEY          (your license key)
  5. CUSTOMER_CODE        (your customer code)

                         WHEN YOU GET THEM (5 MINUTES)

═══════════════════════════════════════════════════════════════════════════════

STEP 1: Open .env file
        $ nano .env

STEP 2: Replace 5 lines with Blue Dart values
        BLUEDART_API_KEY=<new_value>
        BLUEDART_API_SECRET=<new_value>
        BLUEDART_LOGIN_ID=<new_value>
        BLUEDART_LICENSE_KEY=<new_value>
        BLUEDART_CUSTOMER_CODE=<new_value>

STEP 3: Save file
        Ctrl+X, Y, Enter

STEP 4: Test
        $ node test-bluedart.js
        ✓ All checks should pass

STEP 5: Start server
        $ npm run dev
        ✓ Should see: Server running on 9000

STEP 6: Deploy!
        ✓ You're live with real Blue Dart
